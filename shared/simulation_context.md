# Gucci HRM Simulation Context

The learner plays the role of Group Global Organization Development Director.

The learner must design a Group-level leadership system that:
1. Codifies shared Group DNA.
2. Evaluates and develops leaders using 360 feedback and coaching.
3. Cascades the model across regions without damaging brand identities.

The competency framework includes:
- Vision
- Entrepreneurship
- Passion
- Trust

Core deliverables:
- Group DNA principles
- Leadership competency framework
- Behavior indicators across emerging, proficient, and role-model levels
- 360 feedback blueprint
- Coaching program
- Talent mobility and succession logic
- Regional rollout and train-the-trainer plan
- Measurement and adoption plan
